场景：你刚和一位投资人开完会，跟 Claude 说："帮我写封 follow-up 邮件。"

❌ 没 Context · General 答案
"您好，

非常感谢您今天抽时间和我交流。今天聊得很有收获，针对您提到的几个问题，我会尽快整理一份补充材料发给您。

如有任何问题随时联系。

祝好"

→ 你还得花 10 分钟告诉 AI：他是谁、聊了什么、他关心什么

✅ 有 Context · Personal 答案
"X 总，

今天聊到您关注的 hardware moat 三个点，整理了一份补充材料。Ken 之前在 OPPO 做的双系统架构能直接 map 到我们这边——上次您问的功耗问题在 Page 4。

另外我答应给您发的 memory 架构图，附件里。

等您下周从香港回来约个深聊。"

→ 一句话，3 秒，直接能发

两个答案的差距 = Context 的差距。
AI 的能力没区别 — 都是 Claude。差的是 AI 认不认识你、知不知道你今天和谁聊过、记不记得你之前答应过什么。

Context 决定了你拿到一个真正懂你的 personal 答案，
还是一个全网通用的 general 答案。

Context 的核心

决定 AI 答案质量的，
是 Why，不是 What 和 How。
